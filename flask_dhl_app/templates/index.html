About

This program is a DHL package tracking application built with Flask. It allows users to enter a tracking number and retrieve detailed tracking information about their packages.

The application interacts with the DHL API to fetch real-time tracking data for a given tracking number. It displays the current status, location, and delivery information of the package. The application also stores the tracking data in a MySQL database for future reference.

Key Features:

  • Track DHL packages by entering a tracking number.
  • Retrieve real-time tracking information, including status, location, and delivery details.
  • Store tracking data in a MySQL database for easy access and historical tracking.
  • Display a user-friendly interface for tracking package progress.
  • Provides a responsive and intuitive user experience.

This program utilizes Flask as the web framework, MySQL as the database management system, and the DHL API for retrieving tracking information. It follows best practices for code organization and uses HTML templates to render the user interface.

Disclaimer: This application is my personal project and is not affiliated with DHL. The tracking data displayed may not be real and is used for illustrative purposes only.
